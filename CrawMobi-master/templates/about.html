O que é o CrawMobi?

O CrawMobi é uma aplicação para obtenção de um conjunto de dados a partir de nomes de modelos dos aparelhos dados como entrada. Obtenha seu conjunto de dados com mais de 20 informações de aparelhos desejados, de maneira simples!

Robusta

As fontes utilizadas para obtenção dos dados são confiáveis e bem conhecidas no ambiante virtual, no que diz respeito a comparação de smartphones. São analisados os dados de cada fonte e a aplicação decide qual o mais confiável para compor sua base de dados. A lista de sites é composta basicamente de três sites: Kimovil, PhoneArena e GSMArena. Para o preço ser mais realista e acompanhar a realidade dos valores na moeda local, o site utilizado para sua obtenção é o Zoom, pegando o menor preço disponível no país.
